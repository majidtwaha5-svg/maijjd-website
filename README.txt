
Welcome to Maijjd Website + Maijd Software Suite

This ZIP includes:
- Website Frontend (React + Tailwind)
- Backend API (Node.js + Express)
- All software folders (System, CRM, Dev Tools, etc.)
- Auto-update & unlimited version structure
- Placeholder docs + installation setup

To get started:
1. Open the frontend_maijjd folder and run: npm install && npm start
2. Open the backend_maijjd folder and run: npm install && node server.js
3. Explore maijd_software/* for categorized modules

Enjoy the powerful Maijd Software experience.
