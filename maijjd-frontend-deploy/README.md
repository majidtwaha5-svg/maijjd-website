### How to Deploy:
1. **Go to** [vercel.com](https://vercel.com)
2. **Click "New Project"**
3. **Select "Upload Files"** (not GitHub)
4. **Drag and drop** this entire folder
5. **Set project name**: `maijjd-frontend-manual`
6. **Click "Deploy"**
