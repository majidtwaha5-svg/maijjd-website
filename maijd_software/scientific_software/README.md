## Installation
```bash
# Quick install
curl -sSL https://maijd.software/install/scientific | bash

# Docker install
docker run -d --name maijd-scientific maijd/scientific-software:latest

# Manual install
git clone https://github.com/maijd/scientific-software.git
cd scientific-software
./install.sh
```
