## Installation
```bash
# Quick install
curl -sSL https://maijd.software/install/apps | bash

# Docker install
docker run -d --name maijd-apps maijd/application-software:latest

# Manual install
git clone https://github.com/maijd/application-software.git
cd application-software
./install.sh
```
