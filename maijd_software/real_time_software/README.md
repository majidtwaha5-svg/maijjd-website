## Installation
```bash
# Quick install
curl -sSL https://maijd.software/install/realtime | bash

# Docker install
docker run -d --name maijd-realtime maijd/realtime-software:latest

# Manual install
git clone https://github.com/maijd/realtime-software.git
cd realtime-software
./install.sh
```
